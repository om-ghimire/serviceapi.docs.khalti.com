## Type: Multi Step API

### User Detail Fetch API

- **Request URL:** {{base_url}}/api/servicegroup/details/hashcnet-v2/
- **Request Method:** POST

#### Service Params

<pre><code class="json">
{
    "token": "token",
    "reference": "unique reference",
    "username": "jeetenaug4",
    "service_slug": "\ service_slug\ "  // Service slug from above list
}
</code></pre>

## Payment API

**Request URL:** {{base_url}}/api/servicegroup/commit/hashcnet-v2/

**Request Method:** POST

**Service Params:** 
<pre><code class="json">
{  
	"session id": 6466, \\ session id from detail fetch API     
    “reference”: “unique identifier”,  
	"token": "Test token",  
	"package code": “9”,  
    “duration code”:”1”,  
    “amount": 999.00  
}
</code></pre>
